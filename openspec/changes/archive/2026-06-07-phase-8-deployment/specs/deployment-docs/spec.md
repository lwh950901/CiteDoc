### Requirements

#### 1. README 完整性
- MUST 包含项目简介、功能特性、技术栈
- MUST 包含本地开发步骤（含环境变量配置）
- MUST 包含部署到 Vercel + Supabase 的分步指引
- MUST 包含项目架构图（ASCII）
- MUST 包含关键设计说明
- MUST 包含 API 端点列表和 SSE 事件格式

#### 2. Metadata
- layout.tsx MUST 设置英文 title + Open Graph 元数据
- description MUST 准确描述项目核心价值（溯源）

#### 3. 路线图
- Phase 1-8 MUST 全部标记为完成
